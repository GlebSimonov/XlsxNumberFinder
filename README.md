## Зависимости

Для работы проекта требуется:
- Java 17 или выше
- Maven

## Шаги для сборки и запуска проекта

### 1. Сборка проекта

```bash
mvn clean install
```

### 2. Запуск проекта

```bash
mvn spring-boot:run
```

### 3. Использование Swagger
После запуска сервиса откройте браузер и перейдите по следующему URL: http://localhost:8080/swagger-ui.html
